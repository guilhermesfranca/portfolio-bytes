Estou muito feliz em compartilhar a conclusão de um projeto especial: meu portfólio pessoal! 🚀

Este foi o projeto de encerramento do primeiro módulo no meu bootcamp de desenvolvimento na Bytes4Future, onde tivemos o desafio de criar um portfólio completo, partindo do zero.

A jornada começou no Figma, onde desenhei toda a interface e experiência do usuário (UI/UX). Em seguida passei para o código para dar vida ao design, utilizando HTML5 e CSS3 para construir cada detalhe.

O resultado é um portfólio moderno e interativo, com foco em:

Responsividade: O layout se adapta perfeitamente a qualquer dispositivo, de celulares a desktops.
Animações Avançadas: Implementei transições e efeitos dinâmicos, como o "card-flip" 3D na seção de idiomas e as interações nos cards de projetos, para criar uma experiência mais engajante.
Código Limpo e Estruturado: Uma base sólida que demonstra minhas habilidades em front-end.
